Hi! This is a Wordle project I'm constantly tinkering with. If you're unfamiliar with Wordle, here are the rules: you have to guess the 5-letter word the computer has chosen. Each letter is judged individually - if the letter is in the word, but in the wrong place, it will be highlighted amber. If it is a right letter in the right place, it will be highlighted green. You have six guesses to figure out the word - good luck! :)
